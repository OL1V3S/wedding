# Wedding Website

Hosted at https://ol1v3s.github.io/wedding/

This is my official personal wedding website.  
It includes event details, venue information, registry links, livestream access, and an RSVP form.

The site is fully bilingual (English / Spanish).


---

## Features

- English / Spanish language toggle
- RSVP form (Formspree)
- Embedded Google Map for venue directions
- Livestream Link 
- FAQ section

---

## Tech Stack

- **HTML5**
- **CSS3** (custom, no framework)
- **JavaScript (ES Modules)**
- **Vite** (dev server & build)
- **GSAP + ScrollTrigger** (animations)
- **Formspree** (RSVP handling)

No backend required.

---


## Running Locally

```bash
npm install
npm run dev
```
---
## RSVP Handling

RSVP submissions are handled via Formspree and include:

- Family name
- Attendance confirmation
- Guest count
- Dietary restrictions / allergies
- No account is required for guests
